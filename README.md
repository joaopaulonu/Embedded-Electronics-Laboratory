## 📄 Overview

This repository contains a comprehensive collection of **electronic circuits and laboratory exercises** developed for the **Laboratório de Eletrônica Embarcada (Embedded Electronics Lab)** course. 

The core of this project is the use of **LTspice** for high-fidelity circuit simulation, documenting the behavior of discrete components and integrated systems. It serves as a technical log of circuit design, waveform analysis, and hardware fundamentals.

---

## 🧠 Why LTspice?

**LTspice** is a high-performance SPICE simulation software, schematic capture, and waveform viewer with enhancements and models for easing the simulation of analog circuits.

### Using LTspice allows for:

- **Transient Analysis:** Observing how circuits behave over time.
- **AC Analysis:** Evaluating frequency response and gain in amplifiers.
- **Component Validation:** Testing diodes, transistors (BJT), and Op-Amps before physical prototyping.
- **Power Electronics:** Simulating switching behaviors of SCRs and TRIACs.

---

## 🚀 Key Skills Demonstrated

The simulations and reports in this repository demonstrate:

- **Analog Design:** Understanding semiconductor biasing and signal rectification.
- **Signal Conditioning:** Using Operational Amplifiers for summing, filtering, and gain adjustment.
- **Power Control:** Implementing AC control using thyristors (SCR/TRIAC).
- **Data Conversion:** Designing R-2R ladders and active filters for Digital-to-Analog conversion (DAC).
- **Technical Analysis:** Comparing theoretical calculations with simulated waveforms (Vpp, Vavg, Ripple).

## 💡 Concepts Covered

- **Rectification & Filtering:** Converting AC to DC with ripple reduction using capacitors.
- **Transistor States:** Managing Cut-off, Saturation, and Active regions in BJTs.
- **Thyristor Control:** Triggering mechanisms for SCRs and phase control.
- **Mathematical Circuits:** Using Op-Amps for summing frequencies and Fourier Theorem visualization.
- **Interfacing:** Bridging digital logic with analog outputs.
